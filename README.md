# Solé Yoga — San Diego

A landing page for a fictitious yoga studio based in Ocean Beach, San Diego.

## About

Solé Yoga is a sanctuary for modern seekers, where ancient practice meets coastal serenity. This landing page showcases:

- **Hero Section** — Immersive introduction with coastal imagery
- **Classes** — Three featured yoga offerings (Vinyasa Flow, Restorative, Power Yoga)
- **About** — Studio story with overlapping image composition
- **Schedule** — Weekly class schedule with instructor details
- **Testimonials** — Member testimonials with elegant typography
- **CTA Section** — Call-to-action for new student offer
- **Footer** — Contact info and navigation links

## Design Approach

- **Aesthetic**: Organic, coastal-inspired with warm earth tones
- **Colors**: Sand, sage green, terracotta, warm charcoal
- **Typography**: Cormorant Garamond (display) + DM Sans (body)
- **Features**: Animated organic blob shapes, hover interactions, responsive design
